# CyberScan v2

CyberScan v2 is a GUI-based TCP port scanner built with Python and Tkinter.

The project began as a command-line port scanner and evolved into a desktop application while learning networking, socket programming, threading, and GUI development.

## Features

* Hostname to IP Resolution
* TCP Port Scanning
* Service Detection (HTTP, HTTPS, SSH, FTP, etc.)
* Dark-Themed GUI
* Real-Time Progress Bar
* Scan Duration Tracking
* Open Port Reporting
* Multi-threaded Architecture

## Technologies Used

* Python
* Tkinter
* Socket Programming
* Threading
* Git
* GitHub
